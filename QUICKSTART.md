# 快速开始指南

## 🎯 项目说明

这是一个**数字人格分身生成器**，通过12道脑洞大开的问题，为你生成专属的MBTI人格分析、虚拟形象、职业预测和社交文案。

## ⚡ 三步启动

### 第一步：安装依赖
```bash
npm install
```

### 第二步：启动开发服务器
```bash
npm run dev
```

### 第三步：打开浏览器
访问 **http://localhost:5173**

## 🎮 使用流程

1. **欢迎页面** - 了解功能，点击"开始测试"
2. **答题环节** - 回答12个趣味问题（约3分钟）
3. **查看结果** - 获得你的数字人格分身：
   - MBTI性格类型
   - 虚拟形象（emoji + 风格）
   - 性格标签
   - 未来职业预测
   - 朋友圈/小红书文案

## 🌟 特色亮点

### 脑洞问题示例
- 🌙 凌晨三点醒来会做什么？
- 👻 如果能隐身想偷听谁说话？
- 🚀 想去哪个星球干什么？
- 💰 中1000万彩票的第一反应？

### 16种人格类型
完整覆盖所有MBTI类型，每种都有独特的：
- Emoji虚拟形象
- 4个性格标签
- 职业发展方向
- 社交分享文案

## 🛠️ 技术栈

- **React 18** + **TypeScript**
- **Vite** 构建工具
- **Framer Motion** 动画库
- 响应式设计，支持移动端

## 📱 分享功能

测试结果页面可以一键复制：
- 💬 朋友圈风格的短文案
- 📕 小红书风格的种草笔记

## 💡 应用场景

- ✅ 大学生社团活动/破冰游戏
- ✅ 社交媒体分享互动
- ✅ 自我认知和职业规划参考
- ✅ 朋友间的趣味话题

## 🔧 自定义修改

### 添加新问题
编辑 `src/data/questions.ts`

### 修改人格数据
编辑 `src/data/personalityData.ts`

### 调整样式
编辑 `src/index.css`
